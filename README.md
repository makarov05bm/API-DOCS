## package.json commands
```js
npm init -y
```
### packge.json configuration
```js
    "start": "NODE_ENV=production node server",
    "dev": "nodemon server"
```
### git initialisation
- git init in the root
- add a `gitignore` file in the root and add `.env` file and `node_modules/`
